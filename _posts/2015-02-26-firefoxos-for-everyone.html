Kaustav hosted an event for Firefox OS keeping in mind the non-developers - "FirefoxOS for Everyone."

Firefox OS is an ecosystem in itself. Everyone, not just developers, can contribute to it, and help the ecosystem grow. The aim of this event is to help participants learn about Firefox OS from a user point-of-view. It will enable participants to talk about Firefox OS to non-developers, by understanding the core concepts, and learning to explain them in simple words.

More info and slides on the blog. We'll be hosting this workshop once in every month. Keep an eye on our events page for the date.
